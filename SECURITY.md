# Security Policy

DigitalServerHost (DSH) is committed to providing a secure and resilient infrastructure. We take security vulnerabilities very seriously.

## Reporting a Vulnerability

Please report security vulnerabilities directly to **support@digitalserverhost.com**.

Do not disclose security vulnerabilities publicly until they have been addressed. We aim to respond to all reports as soon as possible.

## Disclosure Policy

Once a vulnerability has been fixed, we will coordinate with the reporter to disclose the issue in a way that protects our users and infrastructure.
